# Hogwart Heroes CDN Solution 

## Requirement

The Hogwarts Foundation, a non-profit organization funded by Wizard Governments worldwide, has discovered a peculiar new interest among its fantastic beasts: a deep fascination with heroes, especially the Avengers! To channel this heroic enthusiasm, Newt Scamander, who runs the foundation, wants to create the "Heroes" This will be a dynamic web platform and database designed to catalog heroes from all worlds and rank them with a unique scoring system. He is asking us to manage the infrastructure for this ambitious project over Hogwarts AWS in High Availability manner

## Implementation 

- Manage data with simple JSON instead of database
- Create a Heroes Static File Generator
- Using AWS S3 for Heroes Contents
- Deploy with AWS Amplify as CDN Solution
